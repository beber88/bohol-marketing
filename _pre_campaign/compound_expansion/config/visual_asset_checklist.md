# Visual Asset Checklist - Compound Expansion

Bar, these are the renders and photos we need to launch the compound campaign.
Priority order: top items are critical, bottom items are nice-to-have.

## CRITICAL (Cannot launch without these)

### 1. Compound Aerial Render
- All 8 villa positions visible from above (A-H)
- Show the compound layout, paths/roads between villas
- Include beach/ocean in frame for context
- This is THE hero image for all marketing
- Format: High-res JPG/PNG, minimum 3000px wide

### 2. Updated Site Plan / Masterplan
- Architectural site plan layout showing all 8 villa positions
- Mark A+B as "Sold", C+D as "Phase 1", E-H as "Phase 2"
- Show distances (to beach, between villas, to JW Marriott)
- Show any common areas, roads, parking
- Format: Clean vector or high-res render

### 3. Villa E/F/G Exterior Renders (same design as C/D)
- Front facade (street view) - 1 per villa minimum
- Rear/pool side view - 1 per villa minimum
- Can use same render with minor angle variations since specs match C/D
- Format: High-res JPG/PNG, minimum 2400px wide

### 4. Villa H Exterior Renders (unique design)
- Front facade - minimum 2 angles
- Rear/pool side - minimum 1 angle
- Any distinctive design features highlighted
- Format: High-res JPG/PNG, minimum 2400px wide
