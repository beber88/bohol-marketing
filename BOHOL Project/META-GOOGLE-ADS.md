## AD TESTING MATRIX

| Variable | Test A | Test B | Winner Criteria |
|---|---|---|---|
| Hook | ROI angle | Lifestyle angle | CTR > 2% |
| CTA | "Learn More" | "Send Message" | CPL < $30 |
| Image | Exterior render | Pool/sea view | Engagement rate |
| Audience | Israel | UAE | Cost per lead |
| Format | Single image | Carousel | Lead quality |

**Testing Rules:**
- Run each variation minimum 3 days before judging
- Minimum 1,000 impressions before making decisions
- Never change image + copy in same test (isolate variables)
- Kill any ad with CPM above $18 or CTR below 0.8%
- Scale any ad with CTR above 3% (double budget)
