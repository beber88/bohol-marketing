# PANGLAO PRIME VILLAS - MASTER HOOK LIBRARY

## What is a Hook?
The first 3 seconds of a video or first line of an ad.
It must stop the scroll. It must create a question.
It must make the viewer feel something before they think.

## HOOK TESTING PROTOCOL

Run each hook for 3 days minimum before judging performance.
Kill hook if CPM above $15 or CTR below 1.5%.
Scale hook if CTR above 3% or CPL below $30.
Always test 2 hooks per ad set simultaneously (A/B).
Never change image and copy at the same time (isolate the variable).
