# Facebook Group Research Guide
## Panglao Prime Villas - Target Groups by Market

> **How to use this guide:**
> 1. Copy each search query into Facebook's search bar
> 2. Filter by "Groups"
> 3. Join groups with 1,000+ members
> 4. After joining, click "Sync Groups" in the Autoposter to import them
> 5. Classify each group by market/type/language in the Groups Library

## GROUP QUALITY CHECKLIST

Before joining a group, check:

- [ ] **5,000+ members** (exceptions for niche/local groups)
- [ ] **Active posts** (at least 3-5 posts per week)
- [ ] **Relevant content** (investment discussions, not just spam)
- [ ] **Allows posting** (some groups are very restrictive)
- [ ] **Not a competitor's group** (avoid groups run by competing developers)
- [ ] **Public or easy approval** (private groups that never approve = waste of time)

## AFTER JOINING

1. **Wait 24-48 hours** before posting (don't spam immediately after joining)
2. **Read the group rules** first
3. **Start with value posts** (education, market data) before any promotional content
4. **Engage with existing posts** (comment, like) before posting your own
5. **Use the Autoposter's "Posting Rules" field** to note each group's specific rules
6. **Set quality score** (1-5 stars) based on group activity and relevance
